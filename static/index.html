🔑 忘记密码？

本应用数据存储在本地，无法通过邮件重置密码。

请联系管理员（通过 SSH 登录设备）执行以下命令重置 admin 密码：

cd ~/diary_web
python3 -c "
  import json, hashlib, base64, os
  salt = os.urandom(16)
  h = hashlib.pbkdf2_hmac('sha256', b'新密码', salt, 600000)
  d = json.load(open('config/users.json'))
  d['admin']['password_hash'] = base64.b64encode(salt+h).decode()
  d['admin']['password_changed'] = False
  json.dump(d, open('config/users.json','w'), indent=2, ensure_ascii=False)
"

将上面代码中的新密码替换为你想要设置的新密码。
